Employee API Manager

Project Overview

The Employee API Manager is a Django-based application designed to manage employees, their assets, documents, salary slips, and more.

Features

  • User Authentication: JWT and session-based authentication.
  • Employee Management: Add, edit, delete, and retrieve employee details.
  • Document Uploads: Attach and manage documents related to employees.

Installation & Setup

git clone https://github.com/your-repo/employee-api-manager.git
cd employee-api-manager
python -m venv env
source env/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver

Usage Examples

Registering an Employee

import requests

data = {
    'name': 'John Doe',
    'email': 'john@example.com',
}

response = requests.post('http://127.0.0.1:8000/api/employees/', json=data)
print(response.json())
